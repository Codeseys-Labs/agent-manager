# agent-manager Phase 1 MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the Phase 1 MVP — a CLI that reads TOML config, resolves profiles, imports/exports Claude Code configs, syncs via git, and detects drift.

**Architecture:** Layered Core + Adapter Extensions (ADR-0001). 4 core entities (Servers, Instructions, Skills, Profiles) with `[adapters.<name>]` escape hatches. Git-backed everything (ADR-0002). Single adapter (Claude Code) for Phase 1.

**Tech Stack:** Bun, TypeScript, citty (CLI), @clack/prompts (wizards), Zod (validation), @iarna/toml (config), isomorphic-git (sync), chalk (output), Biome (lint)
